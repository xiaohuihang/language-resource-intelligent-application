# language-resource-intelligent-application
Language Resource Construction and Intelligent Applications
This repository contains experimental code for the course Language Resource Construction and Intelligent Applications. It demonstrates the application of classic machine learning models to fundamental NLP tasks, using self-constructed datasets.

Contents
API
Sample programs for accessing both local and online DeepSeek models, illustrating how to integrate large language models into applications.

classification
Implementation of traditional classifiers for sentiment analysis on a self-built dataset. Includes:
Decision Tree
Maximum Entropy (MaxEnt)
Naïve Bayes
k-Nearest Neighbors (kNN)
Both code and the dataset are provided.

sequencelabel
Sequence labeling for part-of-speech tagging using:
Conditional Random Fields (CRF)
Hidden Markov Models (HMM)
Maximum Entropy (MaxEnt)
Each model is implemented from scratch or using standard libraries, with training and evaluation scripts.

These materials are intended for educational purposes, showcasing how classic machine learning approaches can be effectively applied to language resource construction and intelligent language processing tasks.
Feel free to explore, modify, and learn from the code.
